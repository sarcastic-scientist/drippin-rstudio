# Drippin Dark — RStudio Theme

A dark RStudio theme built for legibility without going blind. 

The palette avoids red-green-only distinctions (safe for deuteranopia and protanopia). Luminance contrast is used alongside hue so all elements remain distinguishable
across the most common types of colour vision deficiency. Syntax highlighting is aimed primarily at R code, R Markdown, and Quarto documents. It is recommended to use **Drippin** with a nerd font.
